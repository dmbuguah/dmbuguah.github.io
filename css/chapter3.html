ID and Attribute Selector Specificity

  • Its good to note that every inline declaration has a specificity of 1,0,0,0
  • Universal selectors have a specificity of zero hence hence wins when applied to elements with other specificity.
    This is a disasvantage of using universal selectors blindly

Cascade

  • If two rules apply to an element and one is marked !important,the one marked !important wins
  • If two rules have exactly the same weight, origin, and specificity, then the one that occurs later in the style sheet wins out.
